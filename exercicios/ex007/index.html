Nessa frase temos um termo em NEGRITO usando a tag B (não semânica).

Nessa frase, temo um termo em DESTAQUE usando a tag STRONG (semânica).

Itálico / Ênfase

Nesta frase, temos um termo em ITÁLICO usando a tag I (não semântica).

Nesta frase, temos um termo em ÊNFASE usando a tag EM (semânica).

Texto Marcado

Nessa frase, temos um texto marcado usando a tag MARK.

Esse é outro exemplo de marcação de texto.

Podemos também deixar todas as tags MARK com a cor já pré-definida add um style no cabeçalho "head".

Textos grandes e pequenos

Estamos criando um texto grande usando a tag BIG, porém, essa tag está obsoleto e para isso deve se usar o CSS.

E aqui temos um texto pequeno usando a tag SMALL que ainda sim está ativa.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele pode ser lido mas não considerado. Utilizamos a tag DEL.

Texto Iserido

Podemos marca um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizamos a tag INS.

Existe também a opção de sublinha um texto uando a tag U (não semântica).

Texto Sobrescrito

Para inserir coisa do tipo x50+5

Usamos a tag SUP.

Texto Subscrito

Para inserir coisas do tipo H2O

Usamos a tag SUB
